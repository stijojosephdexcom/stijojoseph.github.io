DMA — Direct Memory Access

DMA is a hardware courier that moves data between peripherals and memory while the CPU sleeps or does real work.

01. What it is

DMA (Direct Memory Access) is a peripheral that performs memory copies and peripheral-to-memory transfers autonomously. You configure source, destination, length and trigger source; the DMA controller does the rest, raising an interrupt only when the whole transfer is done (or half done).

Without DMA, every byte from a UART or ADC must be moved by the CPU in an ISR — fine at 9600 baud, hopeless at 12 Mbps. With DMA, an MCU running at 16 MHz can sustain audio I²S or USB throughput while the CPU handles application logic.

02. How it works

A DMA controller has multiple channels or streams, each programmable with:

  • Source and destination addresses (memory or peripheral register).
  • Direction — peripheral→memory, memory→peripheral, or memory→memory.
  • Transfer size — byte / half-word / word.
  • Mode — single, circular (auto-restart), or burst (multiple words per request).
  • Trigger — software, or a peripheral request line (SPI RX ready, UART TX empty, ADC EOC).

On Cortex-M7 systems with a cache, you must invalidate or clean cache lines around DMA buffers, otherwise the CPU sees stale data.

04. Real-world examples

  • Audio I²S — circular DMA streams 48 kHz × 16-bit × 2-channel samples to a DAC with zero CPU cycles per sample.
  • UART RX with idle-line detection — DMA fills a ring buffer; an idle interrupt signals end-of-packet.
  • ADC sweep — DMA copies multiple ADC channels into a buffer; the CPU is interrupted only on full conversion.
  • SPI display refresh — DMA pumps the entire framebuffer out over SPI while the CPU renders the next frame.

05. Code snippet

C — UART circular DMA receive 
#define RX_LEN 256
static volatile uint8_t rxBuf[RX_LEN];

void configure(void) {
    /* Enable idle-line interrupt on USART1 */
    __HAL_UART_ENABLE_IT(&huart1, UART_IT_IDLE);
    HAL_UART_Receive_DMA(&huart1, (uint8_t*)rxBuf, RX_LEN);
}

void USART1_IRQHandler(void) {
    if (__HAL_UART_GET_FLAG(&huart1, UART_FLAG_IDLE)) {
        __HAL_UART_CLEAR_IDLEFLAG(&huart1);
        uint16_t received = RX_LEN -
            __HAL_DMA_GET_COUNTER(huart1.hdmarx);
        processPacket((uint8_t*)rxBuf, received);
        HAL_UART_Receive_DMA(&huart1,
            (uint8_t*)rxBuf, RX_LEN);
    }
    HAL_UART_IRQHandler(&huart1);
}

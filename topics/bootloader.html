Bootloader

A bootloader is a tiny, trusted helper program that decides which firmware runs — and that gives you safe over-the-air updates.

01. What it is

A bootloader is a small, immutable program in the lowest sector(s) of Flash. After reset the CPU always lands in the bootloader. Its job is to:

  1. Decide whether to enter "update" mode (button held, magic byte set, no valid app, …).
  2. Optionally accept a new firmware image over UART, USB, BLE, or LoRa.
  3. Validate the candidate image (CRC, signature, version).
  4. Jump to the application's reset vector.

02. How it works

The Flash is laid out in fixed regions:

  • Bootloader — first ~16 KB, locked.
  • Application slot A — currently running firmware.
  • Application slot B — the staging area for a new image (A/B updates).
  • Metadata — pointers to which slot is active, signature, CRC.

To flash a new image the bootloader erases the destination sector(s), writes the new bytes, recomputes the CRC over the whole image and only swaps the "active slot" pointer if validation passes. If the new firmware fails to "check in" within a few seconds, the bootloader rolls back — that's how OTA updates can survive a power loss mid-flash.

04. Real-world examples

  • STM32 system bootloader in ROM — DFU over USB or UART, used to flash a brand-new chip.
  • MCUboot — open-source, signature-verifying bootloader used by Zephyr, Mynewt, and PX4.
  • Tesla & modern cars — A/B partition OTAs with rollback if the new firmware fails to boot.
  • Arduino bootloader (optiboot) — tiny serial bootloader on AVR that lets you flash via USB-serial.

05. Code snippet

C — jump to application vector 
#define APP_BASE 0x08004000

typedef void (*pFunction)(void);

static void jumpToApp(void) {
    uint32_t appStack = *((uint32_t*)APP_BASE);
    uint32_t appReset = *((uint32_t*)(APP_BASE + 4));

    /* Disable interrupts and re-locate the vector table */
    SCB->VTOR = APP_BASE;
    __set_MSP(appStack);
    pFunction reset = (pFunction)appReset;
    reset(); /* never returns */
}

int main(void) {
    HAL_Init();
    if (imageValid(APP_BASE) && !forceUpdateRequested())
        jumpToApp();
    runUpdateLoop();
}

Microcontroller (MCU)

A microcontroller fuses a CPU, RAM, Flash and I/O peripherals into a single low-power chip — the brain of nearly every embedded device.

01. What it is

A microcontroller (MCU) is a complete computer on a single integrated circuit. Unlike a PC processor that needs external memory and chipsets, an MCU bundles the processor core, program memory (Flash), working memory (SRAM), a clock generator, and a rich set of peripherals — UART, SPI, I²C, timers, ADC, GPIO — all on one die.

The result is a chip that boots in microseconds, costs from a few cents to a few dollars, and can run for years on a coin cell. From your microwave oven to a Mars rover, MCUs are the silent workhorses of the embedded world.

02. How it works

An MCU has three layers you should picture together:

  1. The core — typically an ARM Cortex-M, AVR, RISC-V or ESP32 Tensilica CPU. It executes machine instructions, handles interrupts, and talks to memory over an internal bus matrix.
  2. On-chip memory — Flash holds your firmware, SRAM holds runtime data, and a smaller block of EEPROM (or emulated EEPROM in newer parts) keeps non-volatile config.
  3. Peripherals & pins — every peripheral block is wired to a programmable pin mux. By writing to a register you decide whether pin PA9 is a UART TX line, a PWM output, or a digital input.

Software development looks just like normal C/C++, but linker scripts place each section into its physical memory region, and a vector table at address 0 tells the core where to jump on reset and on each interrupt source.

04. Real-world examples

  • STM32F4 / STM32L4 (ST) — ARM Cortex-M4 with FPU; ubiquitous in industrial, medical and consumer products.
  • ESP32 (Espressif) — dual-core Xtensa with WiFi + BLE; the workhorse of low-cost IoT.
  • RP2040 (Raspberry Pi) — dual Cortex-M0+ with programmable I/O state machines.
  • ATmega328P (Microchip) — the venerable 8-bit AVR inside the original Arduino UNO.
  • nRF52 (Nordic) — Cortex-M4 plus a Bluetooth Low Energy radio, the basis of most BLE wearables.

05. Code snippet

C — bare-metal blink 
#include "stm32f4xx.h"

int main(void) {
    /* 1. Enable GPIOA clock */
    RCC->AHB1ENR |= RCC_AHB1ENR_GPIOAEN;

    /* 2. Configure PA5 as push-pull output */
    GPIOA->MODER |= (1 << (5 * 2));

    while (1) {
        GPIOA->ODR ^= (1 << 5);
        for (volatile int i = 0; i < 200000; i++);
    }
}
